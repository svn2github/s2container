S2Container is a lightweight container which supports DI(Dependency Injection) and AOP(Aspect Oriented Programming). We would like to ask you this question first.


Do you wanna write configuration files or not?


Sure, you don't wanna write them. S2Container employs Convention over Configuration concept, where S2Container automatically set up configuration files appropriately based on the "Convention". And, Quickstart allows you to master S2Containter in 15 minutes.

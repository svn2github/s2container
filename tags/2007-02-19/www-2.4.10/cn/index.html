欢迎光临Seasar项目组

　我们Seasar项目组是以基于开放源代码的组成方式，通过为开发者提供一个能够实现从容开发产品的环境，从而帮助用户和开发者一起分享欢喜和快乐，这是我们的活动宗旨。

Java 项目

S2Container.Java

产品 说明 Resources
Seasar2
(S2Container)		 支持DI (Dependency Injection) 和AOP (Aspect Oriented Programming)的轻量级容器

Presentation.Java: 支持Web应用的设计层的Java项目

产品 说明 Resources
Mayaa 基于HTML的模版文件，据此而形成的WEB前台服务引擎
S2Flex 支持在Flash播放器中用Flex调用S2Container组件的软件库
S2JSF 提供基于HTML的模版引擎，基于JSF的软件框架
S2OpenAMF 支持在Flash播放器中用Flash Remoting调用S2Container组件的软件库
S2Struts 在Struts中调用S2Container组件的软件库
S2Tapestry 在Tapestry中S2Container组件的软件库
Teeda 在JSF实装和DI及AOP机能的基础上提供扩张机能的软件

Persistence.Java: 支持永续化的Java产品

产品 说明 Resources
Kuina JPA (Java Persistence API) 软件框架 准备中
S2Dao 无须用XML定义映射及实装代码的O/R Mapping 软件框架
S2Dxo 支持设计层模型和域模型互相变换的软件库
S2Hibernate 通过S2Container的业务处理(transaction)控制和组合来使用Hibernate为目的的软件库

Communication.Java: 支持分散处理的Java产品

产品 说明 Resources
S2Axis 支持在用Axis远程调用S2Container组件的软件库
S2Remoting 用S2Container进行远程调用的通用基盘软件库
S2RMI 支持用RMI远程调用S2Container组件的软件库

Tools.Java (Eclipse Update Site): 用于开发Java产品的开发工具

产品 说明 Resources
Kijimuna 针对S2Container组件的定义文件(dicon文件)进行编辑/查错等的Eclipse插件
S2JSF插件 针对以S2JSF为基础的模版(HTML)进行编辑/查错的Eclipse插件

AllInOne.Java: 开发Java产品用的工具

产品 产品 Resources
AllInOne 仅此一件就可完成所有开发的软件框架箱
Chura 让应用EJB3、JSF、JPA的开发效率产生飞跃提升的All in One软件包

Sandbox.Java: 正在开发的Java产品

产品 说明 Resources
Akabana 以S2 2.4, S2Flex2为中心、将同RIA有关的软件库集成到一起的项目
Cadhelin 可以在Controller类中定义复数个Action方法，支持PRG模式的设计层软件框架
Cubby 基于JSP的网络应用程序软件框架
DBFlute 自动生成“Dao”和“Entity”，使用“S2Dao”的支持工具
DbLauncher Eclipse 3.1 以上、进行H2等的DB起动的插件
Seasar Eclipse Common 开发Eclipse 插件时可以用到的utility软件库
机器猫莫卡 在测试代码中生成模拟类的生成工具
Kvasir/Sora pluggable的应用程序平台
なでじゃこ 参照日文版成语言「なでしこ」而创作的脚本语言
舞姬 快速启动Tomcat的Eclipse插件
maven-backport175-plugin 以使用Backport175为目的的Maven2插件
Mistral 基于Web应用的简单BTS
S2AnA 用S2Container进行认证 (Authentication)和承认 (Authorization) 软件库
S2Buri 管理WorkState的软件
S2Coffee Tuigwaa和S2Buri共通软件库
S2Continuations 在S2Container中实现"继续"的插件
S2Dao-CodeGen 对应数据库表而自动生成DTO、DAO、dicon的工具
S2DaoPlugin 针对S2Dao开发的Eclipse支援插件
S2Directory 以目录服务器的连接简单化为目的的组件管理机能和，将目录服务器的entry属性及对象联结的O/D Mapping (Object / Directory Mapping) 软件框架
S2Ebi 进行Easy(简单) Business(商用) Imperative(规则) 等一系列处理的软件库
S2Flex2 支持Flex2和S2Container联手运用的软件库
S2Hessian 用Hessian/Burlap远程调用S2Container的软件库
S2Javelin 利用S2Container的AOP结构、输出在S2Container上的应用程序中的动作日志的工具
S2JCA JCA (J2EE Connector Architecture)标准的resource adapter得以使用的JCA Application Server Facilities 实装
S2JCR 用Seasar取得JSR 170 (Java Content Repository)并加以应用为目的的软件库
S2JFace 根据Eclipse中使用的SWT/JFace使得将画面用XML记述成为可能、可以简单地实现从GUI调用Seasar组件的软件库
S2JMS 使非同步消息更加容易的收发为目的的软件库、还有让收信消息绑定在S2Container组件上的软件框架
S2JmxJavelin 利用S2Container的AOP结构、S2Container上的应用程序的动作经由JMX而公开的工具
S2Kestrel S2Container的J2SE1.3对应版
S2Laszlo 在Flash播放器中用Laszlo调用S2Container组件的软件库
S2Mai 使用AOP发送电子邮件的软件库
S2Portlet 为了在JSR 168标准的Portlet上使用S2Container的软件库
S2Sao 使用AOP执行JavaScript逻辑的软件库
S2SqlMap iBATIS SQL Maps的对应软件库
S2TopLink 用S2Container transacting的控制和组合来使用TopLink的软件库
S2Velocity 让Velocity和Seasar联合协作的组件
S2XWork 让泛用命令模式软件框架XWork和S2Container联合协作的软件库
Seasar-genapp 利用S2Container开发，生成项目雏形的Maven插件
Tuigwaa 不懂得HTML标签的人，无需专门知识和特别的环境仅需浏览器就可以制作同数据库联动的Web应用程序的工具
